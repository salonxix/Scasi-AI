# Product Overview

Scasi AI is an AI-powered email management system that integrates with Gmail to provide intelligent email handling capabilities.

## Core Features

- Gmail OAuth integration for secure email access
- AI-powered email classification and prioritization
- Smart email summarization and reply suggestions
- Task extraction from email content
- Interactive AI assistant with chat interface
- Email analytics and burnout tracking dashboard
- RAG (Retrieval-Augmented Generation) for context-aware responses

## AI Capabilities

The system uses multiple AI providers (Groq, Gemini) with intelligent routing and rate limiting. Key AI functions include:

- Email classification by category and urgency
- Content summarization for quick review
- Smart reply generation based on context
- Task extraction and action item identification
- Conversational assistant with email context awareness

## User Experience

Users authenticate via Google OAuth, granting access to their Gmail. The dashboard provides an overview of email activity, AI-generated insights, and productivity metrics. The assistant sidebar enables natural language interaction with email data.
